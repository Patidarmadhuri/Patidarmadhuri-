## Featured project: Financial Resilience Dashboard

> React + Flask + MongoDB · Deployed on Vercel + Render

The dashboard pulls financial data from MongoDB Atlas and renders 4 live Plotly.js charts — cash conversion trends, long-term debt ratios, a resilience heatmap, and a bubble chart for risk positioning.

**Architecture:** React (Vercel) → Flask REST API (Render) → MongoDB Atlas  
**API endpoint:** `GET /api/charts` returns all chart data as Plotly JSON  
**Designed for:** financial analysts, developers, and non-technical users alike

[View repository](https://github.com/Patidarmadhuri/Financial-Dashboard-Project) · [Open live demo](https://financial-dashboard-project-eta.vercel.app/)
